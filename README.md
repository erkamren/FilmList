# The Filmlist

A IMDB clone built using React.js. It fetches the data from TMDB (The Movie Database) API. 

# Topics

`ReactJS`
`Tailwind CSS(with JIT)`
`TMDB (The Movie Database) API`
`Hero Icons`
`Responsive Design`
`useState and useEffect`
`React Router`
